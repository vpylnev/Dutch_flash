# 📱 Мобильная оптимизация

## ✨ Что изменилось

Приложение теперь **полностью оптимизировано для мобильных устройств**!

## 🎯 Проблема

На мобильных устройствах левая панель (категории) и правая панель (список слов) занимали слишком много места, из-за чего:
- Карточка была слишком маленькой
- Интерфейс выглядел перегруженным
- Неудобная навигация

## ✅ Решение

### Всплывающие боковые панели

На экранах **меньше 768px** (планшеты и телефоны):

1. **Боковые панели скрыты по умолчанию**
   - Освобождают место для карточки
   - Карточка занимает весь экран

2. **Кнопки меню вверху**
   - 📚 **Menu** - открывает категории (слева)
   - 📝 **List** - открывает список слов (справа)

3. **Всплывающие панели**
   - Открываются поверх контента
   - Плавная анимация сдвига
   - Затемненный фон (overlay)

4. **Кнопки закрытия**
   - Кнопка ✕ в каждой панели
   - Клик по затемненному фону
   - Автоматическое закрытие при выборе

## 🎨 Новые элементы

### 1. Кнопки мобильного меню
- Появляются только на мобильных устройствах
- Красивый дизайн с иконками
- Нажатие с эффектом (active state)

### 2. Overlay (затемнение)
- Полупрозрачный черный фон
- Появляется когда открыта панель
- Клик по нему закрывает панели

### 3. Кнопки закрытия (✕)
- В правом верхнем углу каждой панели
- Круглые кнопки с анимацией
- Видны только на мобильных устройствах

## 🚀 Автоматизация

### Автоматическое закрытие панелей

Панели автоматически закрываются когда:
- ✅ Выбрана категория
- ✅ Выбрано слово из списка
- ✅ Клик по затемненному фону
- ✅ Нажата кнопка закрытия ✕

## 📏 Breakpoints (контрольные точки)

| Размер экрана | Режим | Описание |
|--------------|-------|----------|
| > 1200px | Desktop Large | 3 колонки, широкие панели |
| 992px - 1200px | Desktop | 3 колонки, нормальные панели |
| 768px - 992px | Tablet | 3 колонки, узкие панели |
| 480px - 768px | Mobile | 1 колонка + всплывающие панели |
| < 480px | Mobile Small | 1 колонка + очень широкие панели |

## 💪 Улучшения UX

### Карточка на весь экран
- Больше места для слов
- Лучшая читаемость
- Меньше отвлекающих элементов

### Плавные анимации
- Сдвиг панелей: 0.3s
- Появление overlay: 0.3s
- Все transitions плавные

### Удобная навигация
- Одна рука - одно касание
- Кнопки меню в верхней части
- Большие области для нажатия

### Адаптивные размеры
- Карточка: 280px на мобильном
- Шрифт слова: 1.8rem
- Кнопки: полная ширина

## 🎯 Примеры использования

### Открыть категории
1. Нажмите **📚 Menu**
2. Выберите категорию
3. Панель закроется автоматически
4. Карточки обновятся

### Открыть список слов
1. Нажмите **📝 List**
2. Выберите слово
3. Панель закроется автоматически
4. Карточка переключится

### Закрыть панель
- Кнопка **✕** в панели
- Клик по затемненному фону
- Автоматически при выборе
